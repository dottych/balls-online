Gameplay

Use the arrow keys to move. You can now use WASD too. There are buttons for phone devices, although they are not polished.


Hold shift to draw.


Chat by entering something in the chat box, then press enter or press the confirm button.


The checkboxes toggle a semi-transparent background, for better readability. Here's what each checkbox does:



The second row is in the same order, except the last one, which hides points (also called tracks or drawing).

Information

The top left corner shows useful information about the client and the server. The statistics change colors depending on the value.

FPS - frames per second, can flicker a lot


Ping - the ping distance / speed between the client and the server, updates every second


Status - online or offline, only shows offline if the client is connecting or is not connected



Client run - shows how long the client has been running for


Server run - shows how long the server has been running for


Version - shows the version of the client


Platform - shows the platform the client is on


The "Info" tab shows the ID (name) of the local player, X, Y, and the player count.

The "Chat" tab shows the last five messages. The newest one will always start with a yellow color, then fades to white.

The Player list shows all the players.

The center of the game screen shows the notification message. It will then fade out after time passes.

Local players will always have their balls colored blue. The other balls are red. Above the balls are their IDs / names.
